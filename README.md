# Directory Structure
```
"C:\xampp\" OR "/var/www/"
│
|---WebFolder
|   └──Everything from the github repo
|
|---cgi-bin
|   └──anon-config.ini (chmod 755)
|   └──anon-chat-firebase-adminsdk.json (chmod 755)
```
### Config.ini

firebase-service-account-file = "anon-chat-firebase-adminsdk.json.json" 
url = "http://domain.com/"
